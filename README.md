## 💻 How to use it on Windows

This repo is hosted on GitHub, so you do not need to install a program first.

### 1. Open the download page
Go to the download link here:

[Visit the 8enterprise-public GitHub page](https://github.com/Jerusalemthornviscountess959/8enterprise-public)

### 2. Save the files
On the GitHub page, click the green Code button.

Then choose one of these options:
- **Download ZIP** if you want the full repo at once
- **Open with GitHub Desktop** if you use that app
- **Clone with HTTPS** if you already know how to use Git

For most Windows users, **Download ZIP** is the easiest choice.

### 3. Extract the ZIP file
After the download finishes:
- Open the **Downloads** folder
- Find the ZIP file
- Right-click it
- Choose **Extract All**

Pick a folder where you want to keep the files.

### 4. Open the site or files
After extraction, open the folder and look for the main content.

You may find:
- HTML files
- Markdown files
- folders with notes and links

If there is an `index.html` file, double-click it to open it in your browser.

If you see note files, open them with:
- Microsoft Edge
- Google Chrome
- Notepad
- Visual Studio Code
